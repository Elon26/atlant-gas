## Overview

Atlant Gas is a promotional landing page focused on a polished visual design and non-trivial UI interactions.  
The project demonstrates practical frontend solutions for handling user input, client-side validation, and third-party service integration.

## Features

- Fully responsive layout  
- Complex, visually rich UI  
- Contact form with client-side validation  
- Phone number input masking  
- Form submission via EmailJS  
- Modal dialogs (popups)  
- Content slider  
- Spoilers (collapsible sections)  
- Custom React hook for conditional rendering based on viewport width  
- Additional interactive UI elements  

## Tech Stack

- React  
- TypeScript  
- EmailJS  
- CSS / responsive layout
